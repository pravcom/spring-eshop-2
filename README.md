Это простой онлайн магазин который включает в себя следующие функции:
- Авторизация
- Формирование коризны 
- Список товаров
- Оформление заказов
Данный проект полностью реализован на экосистеме Spring. Использовались такие технологии как:
- Spring Boot
- Spring MVC
- Spring Data
- Spring Security
- Spring Websocket
